javascript 고유 기능 - hoisting
        함수 또는 변수의 선언부를 최상단으로 끌어올려주는 기능
        대상은 var과 선언식 함수로 제한(= let, const, 표현식함수는 적용안됨)

        변수 호이스팅 : 변수 선언 전 출력에도 에러 발생하지 않음
        함수 호이스팅 : 선언문 형태의 함수는 미리 호출해도 에러 발생 많음

        잘못된 흐름의 코드이지만 에러를 발생시키지는 않음 
        하지만 호이스팅이 발생할 코드의 작성을 지양하고,
        표현식 함수와 let, const 키워드의 활용 권장
